## 🎯 What is ResumeWise AI?

**ResumeWise AI** is a production-ready resume builder that transforms raw career information into ATS-optimized, interview-winning resumes using **Google Gemini 2.5 Flash**. Unlike generic resume builders, it intelligently rewrites bullet points with strong action verbs, adds measurable impact, and provides real-time ATS scoring—all while preserving your factual information.

### 💡 The Problem

- 📉 **90% of resumes** are rejected by ATS before reaching human recruiters
- ⏰ **6 seconds** - average time recruiters spend on a resume
- 🎯 **Weak bullet points** lacking metrics and impact statements
- 🔄 **Hours wasted** manually formatting and optimizing resumes

### ✅ The Solution

ResumeWise AI provides:
- 🤖 **AI-powered content generation** that rewrites weak descriptions into achievement statements
- 📊 **Real-time ATS scoring** with 5 key metrics and actionable feedback
- 🎨 **3 professional templates** optimized for different industries
- ⚡ **Live inline editing** for instant customization
- 📄 **Multi-format export** - pixel-perfect PDFs and editable DOCX files

## ✨ Core Features

### 🤖 Intelligent AI Content Generation

**Powered by Google Gemini 2.5 Flash**

The AI doesn't just fill templates—it analyzes your experience and transforms it:

| What You Write | What AI Generates (Modern Template) |
|----------------|-------------------------------------|
| "Worked on backend APIs" | "Architected scalable REST APIs handling 50K+ daily requests, reducing response time by 40% through Redis caching" |
| "Led a team" | "Spearheaded cross-functional team of 5 engineers, delivering MVP 2 weeks ahead of schedule" |
| "Improved performance" | "Optimized database queries achieving 3x performance improvement, cutting server costs by $5K/month" |

**Key AI Capabilities:**
- ✅ Strong action verb selection (Architected, Spearheaded, Optimized)
- ✅ Metric enhancement (adds %, numbers, scale where applicable)
- ✅ Template-aware tone (Corporate vs FAANG vs Academic)
- ✅ Keyword optimization for ATS systems
- ✅ Anti-hallucination safeguards (uses only your facts)

### 📊 Advanced ATS Scoring Engine

Get instant feedback with our 5-component scoring system:

| Metric | Weight | What It Analyzes |
|--------|--------|------------------|
| **Skill Match** | 25% | Technical skills alignment with job requirements |
| **Keyword Relevance** | 25% | Industry-specific terminology and action verbs |
| **Role Alignment** | 20% | Experience relevance to target position |
| **Formatting Score** | 15% | ATS-friendly structure (no tables, images, or columns) |
| **Section Completeness** | 15% | Presence of all critical resume sections |

**Score Interpretation:**
- 🟢 **85-100:** Excellent - Highly likely to pass ATS filters
- 🔵 **70-84:** Good - Minor improvements recommended
- 🟡 **55-69:** Fair - Optimization needed for better results
- 🔴 **Below 55:** Needs significant improvement

### 🎨 Three Professional Templates

**1. Professional Template**
- **Best for:** Corporate, consulting, finance, traditional industries
- **Style:** Clean centered layout, gray section dividers
- **Font:** Helvetica, professional spacing
- **Use case:** MNCs, established companies

**2. Modern Template (FAANG-Style)**
- **Best for:** Tech companies, startups, engineering roles
- **Style:** Left-aligned, bold navy headings, green accents
- **Font:** Helvetica with color highlights
- **Signature:** Thick green line under name, triangle bullets (▸)
- **Use case:** Google, Meta, Amazon, Microsoft, Apple

**3. Classic Template**
- **Best for:** Academic, legal, government, traditional professions
- **Style:** Centered headers, serif font, ruled dividers
- **Font:** Times New Roman, formal tone
- **Use case:** Universities, law firms, government agencies

### ⚡ Dual Input Modes

**Manual Entry (5-Step Wizard)**
```
Step 1: Basic Info     → Name, contact, LinkedIn, GitHub

Step 4: Education      → Degrees, institutions, GPAs
```

## 🏗️ Tech Stack

- **Backend:** FastAPI (Python 3.11+), Uvicorn
- **AI Engine:** Google Gemini 2.5 Flash (via Generative AI SDK)
- **Document Generation:** ReportLab (PDF) & python-docx (Word)
- **Frontend:** Vanilla JavaScript, HTML5, CSS3 (Tailwind CSS)

## 📊 ATS Scoring Breakdown
Our system evaluates resumes using the same logic used by modern recruiting software:

* **Skill Match (25%):** Alignment with technical job requirements.
* **Keyword Relevance (25%):** Use of industry-standard terminology and action verbs.
* **Role Alignment (20%):** Matching history with the target job title.
* **Formatting (15%):** Ensuring a clean, parseable structure.
* **Completeness (15%):** Presence of all critical contact and professional sections.

## 📦 Installation & Setup

### Prerequisites
- Python 3.11+
- Google Gemini API Key ([Get one here](https://aistudio.google.com/app/apikey))

### Steps
1. **Clone & Navigate**
   ```bash
   git clone [https://github.com/yourusername/resumewise-ai.git](https://github.com/yourusername/resumewise-ai.git)
   cd resumewise-
   ```

2. **Setup Environment**

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```
3. **Configure API Key**
   
Create a .env file:
```bash
GEMINI_API_KEY=your_key_here
APP_HOST=127.0.0.1
APP_PORT=8000
```

4. **Run**
```bash
uvicorn backend.app.main:app --reload
```
